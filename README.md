# ophix-lang-fr-docs

> Pack de traduction française pour ophix-docs.

French translation pack for [ophix-docs](https://ophix.io).

## Install

```bash
pip install ophix-lang-fr-docs
```

Then set `LANGUAGE_CODE = "fr"` (or `"fr-fr"`) in your server `.env`.

## What is translated

- Admin UI strings: app label, model field labels and help text
- Documentation page: ophix-docs system guide
